Este pacote de extensão oxt para o LibreOffice 5.2+ fornece dicionários para ortografia, tesauro e hifenização para a linguagem veneta (ISO 639-3 vec).

Os arquivos vec.aff, hyph_vec.dic, th_vec_v2.dat e th_vec_v2.idx foram criados do zero por Mauro Trevisan (mauro.trevisan@gmail.com)
