Este paquete de extensións de oxt para LibreOffice 5.2+ fornece dicionarios para ortografía, sinónimos e guionización para a linguaxe venetana (ISO 639-3 vec).

Os ficheiros vec.aff, hyph_vec.dic, th_vec_v2.dat e th_vec_v2.idx foron creados a partir de cero por Mauro Trevisan (mauro.trevisan@gmail.com)
